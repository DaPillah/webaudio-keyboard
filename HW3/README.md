# CompSoundHW3

Run a local server in this folder, open `index.html` in a browser, and click **Play brook** to hear the Web Audio recreation of the SuperCollider patch.

Files:

- `index.html`: demo UI
- `style.css`: visual styling
- `app.js`: Web Audio implementation
- `brown-noise-processor.js`: continuous brown-noise generator
- `blog/index.html`: separate police siren writeup page
- `part2/index.html`: separate Part II workspace
- `part2/police-siren.js`: police siren synthesis for Part II
- `writeup.md`: markdown source for the writeup text
- `assets/brook-signal-flow.svg`: signal flow diagram
- `assets/police-siren-signal-flow.svg`: police siren signal flow diagram
