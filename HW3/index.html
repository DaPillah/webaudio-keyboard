Babbling Brook

A Web Audio translation of the simplified SuperCollider brook patch using continuous brown noise, slow cutoff modulation, and resonant filtering.

Ready. Click play to start the stream.

Reference patch

{
  (
    { RHPF.ar(
        LPF.ar(BrownNoise.ar(), 400),
        LPF.ar(BrownNoise.ar(), 14) * 400 + 500,
        0.03,
        0.1
      )
    }.play
  )
}

The browser version keeps the same idea: noisy input, very slow random motion for the filter cutoff, and a resonant high-pass filter that creates the bubbling texture.

How this version works

  • Generate one continuous brown-noise source for the audio signal.
  • Low-pass that noise at 228 Hz to keep it darker and less windy.
  • Generate a second brown-noise source for slow cutoff motion.
  • Low-pass the modulation at 15 Hz, then map it with a 260 Hz base and 1400 Hz drift.
  • Feed that motion into a narrow resonant high-pass filter with high resonance.
